# Shobhit Verma - Personal Website

A modern, responsive personal website showcasing the professional journey and achievements of Shobhit Verma (TheJugaadGuy).

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Smooth scrolling, animated counters, and hover effects
- **Social Media Integration**: Links to all social platforms
- **Performance Optimized**: Fast loading with minimal dependencies

## Sections

1. **Hero Section**: Introduction with key statistics
2. **About**: Personal background and story
3. **Experience**: Professional journey in IT and SRE
4. **Content Creation**: TheJugaadGuy and TheJugaadSRE brands
5. **Filmmaking**: Award-winning film projects
6. **Travel**: Adventures across 12 countries
7. **Contact**: Ways to connect

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with Flexbox and Grid
- **Vanilla JavaScript**: Interactive functionality
- **Font Awesome**: Icons
- **Google Fonts**: Typography

## Setup

1. Clone or download the files
2. Replace placeholder images:
   - `profile.jpg` - Your professional photo
   - `film1.jpg`, `film2.jpg` - Film project images
3. Update social media links in the contact section
4. Customize content as needed

## File Structure

```
website/
├── index.html          # Main HTML file
├── styles.css          # CSS styling
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Customization

- Update personal information in `index.html`
- Modify colors and styling in `styles.css`
- Add more interactive features in `script.js`
- Replace placeholder images with actual photos

## Deployment

This is a static website that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any web hosting service

Simply upload the files to your hosting provider's web directory.
